🌡️ Temperature Converter

A modern and interactive temperature converter built using HTML, CSS, and JavaScript. This project allows users to easily convert temperatures between different units with a clean UI and real-time results.

✨ Features
🔄 Convert between:
Celsius → Fahrenheit
Fahrenheit → Celsius
Celsius → Kelvin
🎨 Dynamic color indication:
🔴 Hot temperatures
🔵 Cold temperatures
🟢 Normal temperatures
⚡ Instant conversion results
✨ Attractive UI with glassmorphism design
📱 Responsive and user-friendly interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript
🚀 How to Run the Project

Clone the repository

Open the folder

Run the project:
Double-click index.html
OR
Open with any browser

🎯 Functionality

Enter temperature → Choose conversion type → Click Convert

Displays result instantly with correct unit

Color changes based on temperature level

Interactive buttons for selecting conversion mode
